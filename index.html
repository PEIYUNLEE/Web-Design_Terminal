大阪
OSAKA
/ / 大阪市位於日本本州島西南部，是關西地區的工商業和水陸空交通中心，日本的第二大城市。氣候溫和濕潤，四季花木常青、處處河道縱橫。
/ / 如今是日本代表性的國際旅遊觀光城市，同時關西國際機場的落成，使大阪成為旅客前往京都、神戶、四國等關西地區旅遊的交通樞紐。
京都
KYOTO
/ / 自平安京遷都以來，至定都東京期間，京都一直是日本的首都，因此又有「古都」之稱。因為長年的豐富歷史，使得京都市擁有豐富的歷史遺跡，是日本傳統文化的重鎮之一。

Introduction
道頓堀
雖然本來只是條運河
但因1626年劇場小屋搬遷到此處
而發展起來的劇場小鎮
今日的道頓堀是大阪的主要商區之一
沿運河兩岸設有商店街及不少飲食店
河畔的固力果大型霓虹廣告牌
也成為大阪的著名標誌
大阪環球影城
世界4個環球影城主題公園之一
不僅充滿令人興奮與感動的設施、表演
還可以看到許多人氣角色的出現
其中著名的哈利波特魔法世界園區內
矗立著雄偉漂亮的霍格華茲城堡
每年都吸引了眾多遊客遊玩
伏見稻荷
參拜的人在願望實現後
捐獻鳥居以表達謝意
於是位在京都的伏見稻荷大社
以「千本鳥居」而聞名
現在捐獻鳥居的人仍絡繹不絕
範圍因而擴張到主殿後面稻荷山上
據說至今累積有1000座以上
清水寺
旅者首次到京都必去的景點
這座京都最古老的寺院依山而建
名列世界文化遺產之一
無論是春櫻、夏瀑、秋楓、冬雪
都有其獨特的美
也因此人潮總是絡繹不絕
金閣寺
雖然正式名稱是鹿苑寺
但因舍利殿以金色出名 而有此別稱
也被列為世界遺產
在把金閣寺圍起來般的鏡湖池裡
映照著寺院倒影「逆金閣」
可隨著四季氣候變化
欣賞到鹿苑寺不同的樣貌
哲學之道
位於京都市左京區的小徑
1922年開始 在琵琶湖岸側
從天王寺至淨土寺約2公里長的區間
種植了500棵櫻花樹
成為了著名的櫻花隧道
吸引了相當多遊客造訪
大阪城公園
面積有105公頃之大
坐落於大阪城特別史蹟指定地
內有大阪城天守閣、千貫櫓等
著名的歷史建物
也是賞梅和賞櫻的著名景點
一年四季皆能體驗不同的風情
大阪今昔館
位於大阪一座頗具特色的博物館
館內的建築物和街道模型
重現了大阪人過去的生活
並收藏了江戶時代大阪的城市模型
遊客們可以透過陳列品
全面的了解大阪的發展歷程
藍天大廈
大阪市新梅田城內的摩天大樓
總高度高達173公尺
從戶外觀景台「空中庭園」往下望
梅田的高樓大廈群與鬧區近在咫尺
燈光萬千的景象堪稱一絕
成為了大阪地區的地標
黑門市場
超過170年的歷史
不僅是大阪美食的聚集地
也保存了大阪豐富的飲食文化
除了觀光客之外
黑門市場也深受當地人的喜愛
又有「大阪人的廚房」之稱

啟程
/ / 在考完學測的那次寒假，
我們全家規劃了第一次的自助國外旅行，
在眾多自助旅行者推薦下，
地點選在鄰近台灣、食衣住行不會落差太多的日本，
並考量到是首次嘗試自助旅行，
最後決定前往相對便利繁華的大阪、京都體驗六天五夜的日本關西之旅。
行前
/ / 前往一個人生地不熟語言又不通的地方，
自助旅行最重要的無非是行前準備，
最令我深感自豪的是，
這次的旅行完全是由我跟哥哥一起規劃，
只讓爸媽好好享受。
我們花了將近三個禮拜的時間整理了整個行程，
才有了這次完美的旅行。

/ / 抵達日本的第一天，
因為時間已經接近晚上，
我們只去了大阪的心齋橋商圈逛街，
還有大阪的新地標HARUKAS展望台，
在距離地面300公尺的地方，
觀賞大阪屬於我們第一天的城市夜景。

DAY3
/3
/ / 第三天搭著京阪本縣從繁華的大阪轉移到古色古香的京都
第一站來到了日本全國的稻荷神社總本宮——伏見稻荷大社，
說到朱紅色的鳥居，最先聯想到的便是稻荷，
朱紅色被認為是代表稻荷大神豐饒力量的顏色，
崇敬者為表達祈求與感謝之心，於參拜道路供奉鳥居的信仰，
進而形成今日名景「千本鳥居」。
1
/ / 除了著名的賞楓寺廟——東福寺、三十三間堂
第三天的最後一站我們來到了清水寺，
通過一條條在被日式建築包圍的坂道，
我們參觀了高台寺以及八坂神社，
雖然遺憾的是當時清水寺正值整修，沒有看到全貌，
但也深深被雄偉的建築、以及室內的神聖感觸動。
2
/ / 接近傍晚，我們又趕忙去了身為京都最古老的結緣神社——地主神社，
裡面相當知名的便是兩顆戀愛占卜石，
相傳只要許下願望並閉眼成功從一顆走到十公尺遠的另一顆，願望就會成真。
此時，清水寺內又是不同的一番風景，
漫步再花見小路中，
讓我覺得好像身處在夢境之中，才因而看到這麼美的景色。

DAY5
/3
/ / 最後兩天我們又回到了大阪，
在大阪的賞櫻名點之一，就是能代表大阪的大阪城天守閣，
大阪城與名古屋城、熊本城並列歷史上的三大名城。
大阪城曾為豐臣秀吉時的居城，
後被德川家康消滅後，
成為德川幕府控制西日本的重要據點
1
/ / 大阪今昔館收藏了全日本唯一一座江戶時代大阪的城市模型，
館內的建築物和街道模型再現了大阪人過去的生活，
在裡面可以體驗到昔日大阪之景；
大阪車站則是與台北車站相似，
有著一條條繁華的地下街，
也是個shopping的好去處。
2
/ / 通過大阪車站來到梅田大廈，
位於頂層的空中庭園是完全露天的圓形展望台，
可以三百六十度的眺望大阪夜景，
不同於HARUKAS的是他只有一百七十米高，
在大阪的夜晚涼風吹來，
欣賞到的又是另一番風景。
